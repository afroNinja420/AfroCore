# AfroCore

AfroNinja's Core  Spigot plugin.


## Features

* Enable/Disable each command in config.yml
* Debug mode
  * Output events to console


## Commands

* /feed [player|all]
* /heal [player|all]
* /trash
* /help [command]
* /fly [player]
* /flytimer <name> <time + s|m|h>
* /repair|fix [hand|all]
